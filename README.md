サボテンと同様、触れたEntityItemを消失させます。  
サボテンとは違い、触ってもダメージを受けることはありません。  
また、アイテムを持った状態で右クリックすると、持っているアイテムを消失させます。注意してください。  
ただし、DustBoxはDustBoxを消失させません。
